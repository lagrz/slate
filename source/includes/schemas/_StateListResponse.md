## StateListResponse

<a name="schemastatelistresponse"></a>

```json
{
  "states": [
    {
      "code": "string",
      "name": "string"
    }
  ]
}
```

### Properties

Name|Type|Required|Description
---|---|---|---|
states|[[NameCode](#schemanamecode)]|false|An Array of NameCode Objects
- code|string|false|2 letter state code
- name|string|false|State name
