## CountryListResponse

<a name="schemacountrylistresponse"></a>

```json
{
  "countries": [
    {
      "code": "string",
      "name": "string"
    }
  ]
}
```

### Properties

Name|Type|Required|Description
---|---|---|---|
countries|[[NameCode](#schemanamecode)]|false|Array of NameCode Objects
- code|string|false|2 letter country code
- name|string|false|Country name
